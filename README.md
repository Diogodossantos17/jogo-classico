# jogo-clássico
 jogo-clássico.
 
 
 
 Jogo Clássico, é um Web-site desenvolvido para os amantes do desporto Rei "Futebol", com objetivo de mantê-los informados sobre os resultados, calendários de jogos, campeonatos/ligas. Mercados de transferências e muitos mais. Espero poder ajudar no que poder levando a verdadeira diversão e ao amor ao desporto com os objetivos acima plasmados.
 
 
O mesmo Web-site, será construído com as seguintes tecnologias: 

1. HTML
2. CSS
3. JavaScript
4. React (De forma aplicar o conceito SPAS)
5. E, possivelmente será aplicado o React Native para dispositivos móveis.
